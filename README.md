# Week 1

Implemented a paper named [Detection of Diabetes Using Multilayer Perceptron: Proceedings of ICICA 2018](https://www.researchgate.net/profile/Saumendra-Mohapatra/publication/327545512_Detection_of_Diabetes_Using_Multilayer_Perceptron_Proceedings_of_ICICA_2018/links/5cdd2644458515712eae0a3e/Detection-of-Diabetes-Using-Multilayer-Perceptron-Proceedings-of-ICICA-2018.pdf). In this paper, MLP is used for classification of pregnant women. Proposed technique has been applied on the diabetes database of
PIMA for Indian people and is collected from University of California (UCI).
